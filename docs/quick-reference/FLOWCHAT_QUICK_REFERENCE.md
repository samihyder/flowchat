# FlowChat — Quick Reference (Sales & Support)

**Live chat · CRM contacts · LeadSnapper sync**

| Sign in | [digitalbrandcast.com/FlowChat/sign-in](https://www.digitalbrandcast.com/FlowChat/sign-in) |
|---|---|

## Where to go

| Task | Location |
|------|----------|
| Reply to website chats | **Dashboard** |
| LeadSnapper leads | **Dashboard → Contacts** |
| Your status (Online/Busy/Offline) | Bottom of left sidebar |
| Saved reply shortcuts | Type **/** in chat composer |
| New unassigned chats | **Unassigned** (red badge) |

## Availability

| Online | Busy | Offline |
|--------|------|---------|
| Get new chats | Limited routing | No new chats |

## Inbox essentials

- **Assign** conversation → yourself or teammate  
- **Labels** — Hot, Sales, Follow-up  
- **Status** — Open → Pending → Resolved  
- **Snooze** — remind later  
- Visitor panel → open **Contact** in CRM  

## LeadSnapper → CRM flow

```
Google  →  LeadSnapper Export  →  FlowChat Contacts  →  You outreach
```

**Rep:** Export → **Sync qualified (Hot + Warm)**  
**Admin:** Integrations API key + CRM → enable LeadSnapper sync  

## Admin one-time setup

| Step | Path |
|------|------|
| API key | Settings → Integrations → `LeadSnapper` |
| CRM sync | Settings → CRM → ON → Hot + Warm → Provision fields |
| Website chat | Settings → Inboxes → embed code on site |

## Daily checklist

- [ ] Set **Online** at start of day  
- [ ] Clear **Unassigned** queue  
- [ ] Check **Contacts** for new LeadSnapper syncs  
- [ ] Add **notes** after calls  
- [ ] Set **Offline** when done  

## Quick fixes

| Issue | Fix |
|-------|-----|
| Lead missing in CRM | Confirm Hot/Warm + sync ran + CRM sync enabled |
| Widget not on site | Admin: Settings → Inboxes embed + domain allowlist |
| No Marketing menu | Role may be chat-only — ask admin |
