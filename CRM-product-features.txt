Current Product Architecture
- Monorepo workspace for FlowChat applications and shared packages.
- Next.js web application for the agent/admin dashboard.
- Hono API service for authentication, accounts, agents, teams, inboxes, conversations, public widget APIs, and security routes.
- WebSocket service for realtime messaging, typing, subscription, and agent presence.
- Drizzle database package with PostgreSQL schema and migrations.
- Shared type package for cross-application TypeScript contracts.
- Bun-based development workflow.
- Turborepo-style orchestration for multi-package development.
- Environment-based configuration for web, API, WebSocket, database, Redis, object storage, OAuth, and auth secrets.
- OpenAPI endpoint support through the API service.
- Health endpoints for API and WebSocket services.
- CI and deployment planning included in sprint roadmap.
