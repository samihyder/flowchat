# LeadSnapper — Quick Reference (Sales)

**Chrome extension · Find leads on Google → sync to FlowChat CRM**

| Sign in to FlowChat | [digitalbrandcast.com/FlowChat/sign-in](https://www.digitalbrandcast.com/FlowChat/sign-in) |
|---|---|

## 4-step daily workflow

| # | Tab | Action |
|---|-----|--------|
| 1 | **Find** | Pick preset or keyword → Open Google / Maps |
| 2 | **Results** | Click **Scan** → expand leads → **Find owner mobile** |
| 3 | **Pipeline** | **Add to Pipeline** → edit status & notes |
| 4 | **Export** | **Sync qualified (Hot + Warm)** → FlowChat CRM |

## Priority colours

| Hot | Warm | Cold |
|-----|------|------|
| Call first | Good follow-up | Skip CRM sync |

## Flow CRM settings (one-time)

| Setting | Value |
|---------|--------|
| Flow CRM sync | **ON** |
| API base URL | `https://www.digitalbrandcast.com/FlowChat/api` |
| API key | From FlowChat → Settings → Integrations (`fc_live_…`) |

## Admin checklist (FlowChat)

1. Settings → Integrations → Create key **LeadSnapper**
2. Settings → CRM → Enable sync → **Hot + Warm** → **Save & provision fields**

## Quick fixes

| Issue | Fix |
|-------|-----|
| No scan results | Use Google Maps or Places tab, refresh, scan again |
| CRM sync blocked | Turn on sync in LeadSnapper + FlowChat CRM settings |
| No owner mobile | Ask admin for enrichment API keys in Settings |
